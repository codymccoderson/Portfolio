Projects

Solo Dolo

Solo Dolo is a project I helped build in a team of 3 that functionally is a mountain review site for free solo climbers where they are also able to chart their climb and save their route to a database. Solo Dolo was built using Node.js, Express, postgresQL, and HTML Canvas. We used HTML Canvas to allow the user to plot their route up the mountain by clicking on the image of the mountain. With each click, a connecting line is drawn over the image from point to point. Members of the site are allowed to save these routes to a database.

Fast Food Revenge

Fast Food Revenge is a site that calculates the amount of time required to burn the calories you've gained from cheating on your diet, given your preferred method of exercise. It was created using HTML, CSS, and vanilla JavaScript while making several calls to a single API. The user chooses a method of exercise, types their weight, and selects a fast food restaurant on the homepage. On the next page, the entire menu list from that restaurant is displayed in a modal window. After an item is selected, the nutritional information associated with the item is displayed along with the exercise time needed to burn the total calories.
